üîß Step 1: Download and Setup

1 Download the project

Download the Wikipedia to Notion Importer from GitHub and extract it to a folder on your computer.

2 Run the setup script

Double-click the setup script for your operating system:

  • Mac/Linux: Double-click setup.sh or run ./setup.sh in terminal
  • Windows: Double-click setup.bat
‚úÖ The setup script will install everything you need automatically!

üîë Step 2: Create Notion Integration

3 Go to Notion Integrations

Open your web browser and go to:

https://www.notion.so/my-integrations

Or click this button: Open Notion Integrations

4 Create a new integration

Click the "New integration" button and fill out the form:

  • Name: "Wikipedia Importer" (or anything you like)
  • Workspace: Select your workspace
  • Capabilities: Check these boxes:
    • ‚úÖ Read content
    • ‚úÖ Update content
    • ‚úÖ Insert content

Then click "Submit"

5 Copy your integration token

After creating the integration, you'll see a token that starts with secret_

‚ö†Ô∏è Important: Copy this token and keep it safe! You'll need it to use the app.

üìÑ Step 3: Get Your Notion Page ID

6 Go to any Notion page

Open Notion and go to any page where you want the Wikipedia articles to appear. This could be:

  • Your personal workspace
  • A specific project page
  • Any page you have access to
7 Copy the page ID from the URL

Look at the URL in your browser. It will look something like this:

https://notion.so/My-Page-1234567890abcdef

Copy the long string after the last / (in this example: 1234567890abcdef)

üîó Step 4: Grant Access to Your Integration

8 Share the page with your integration

On the same Notion page:

  1. Click "Share" (top right corner)
  2. Click "Add people, emails, groups, or integrations"
  3. Search for "Wikipedia Importer" (or whatever you named it)
  4. Add it with "Can edit" permissions

üéâ Step 5: Launch the App!

9 Start the Wikipedia to Notion Importer

Double-click one of these files to start the app:

  • Mac/Linux: run_gui.py
  • Windows: run_gui.bat
10 Import your first article!

In the app:

  1. Paste a Wikipedia URL (like https://en.wikipedia.org/wiki/Quantum_mechanics)
  2. Enter your Notion token (the secret_... from step 5)
  3. Enter your page ID (from step 7)
  4. Click "Import Article" and watch the magic happen! ‚ú®
